10_coursera
===========

This script takes the information (title, language, start date, 
weeks count, rating) about all the courses on Coursera.org and saves 
them to xlsx.

Usage
-----

```
~$ python3 coursera.py
```

Requirements
------------

- Python >= 3.4
- requests==2.11.1
- openpyxl==2.3.5
- beautifulsoup4==4.5.1
